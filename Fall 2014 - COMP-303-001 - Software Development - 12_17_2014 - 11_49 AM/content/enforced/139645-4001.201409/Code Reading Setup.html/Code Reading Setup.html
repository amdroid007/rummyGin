Projects

We will be using three open-source projects as sample code bases:

  • JHotDraw 5.3
  • jEdit 4.5.2
  • Rhino 1.7 R4

Download and installation

You need to import these codebases into an Eclipse Luna workspace to be able to do the assignments exactly as described. For the more adventurous, you can pull the code directly into Eclipse by finding the source information on each project page, checking out the release, and fiddling around a bit to get the code to properly build into Eclipse. Each project uses a different version control system (CVS, SVN, and Git, respectively). This can require some patience...

For those who prefer a more seamless solution, I've prepared a sanitized version of each code base, ready for import. Download the three codebase zip files on your computer, then, in Eclipse, do File | Import | (General) Existing Projects into Workspace, and go through the motions. The code bases should build automatically.

Running the Applications

Right-click a project and select Run As | Java Application. Pick the appropriate main type:

  • JHotDraw -> JavaDrawApp
  • jEdit -> jEdit
  • Rhino -> ...tools.shell.Main

Surfing the Code

Understanding source code requires quickly navigating between different locations. For this, it pays to be familiar with the powerful code navigation features of IDEs like Eclipse. Make sure you learn to use the Java Views and search features and master some of the top shortcuts.
